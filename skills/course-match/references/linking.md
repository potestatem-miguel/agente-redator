# Linking

Preferir ancoras:

- descritivas
- naturais no fluxo do texto
- coerentes com intencao informacional

Evitar ancoras:

- genericas demais
- promocionais demais
- repetidas no mesmo artigo

Selecao pratica:

- usar no maximo 2 cursos no pacote final
- priorizar o curso de score mais alto, salvo incoerencia editorial
- manter `internal_links` vazio quando nao houver fonte deterministica para esse bloco
